## ✍️ Description

**Awesome Login Web** is an open-source collection of beautifully designed login page templates. Whether you need a classic form, a social login, or a full-page immersive experience — this collection has you covered. Each template is handcrafted with attention to detail, responsive layouts, and real-world usability in mind.

Perfect for developers, designers, and indie makers who want clean, polished, and production-ready authentication pages without starting from scratch.

---

## 🌟 Features

- 📱 **Fully responsive** — looks great on all screen sizes
- 🎨 **Modern UI/UX patterns** — clean, minimal, and professional
- 🔁 **Multiple variations** — classic form, social login, modal, food-themed, and more
- ⚡ **Zero dependencies** — pure HTML, CSS, and JavaScript (some use Tailwind CDN)
- 🛠️ **Easy to customize** — well-structured code with clear class naming
- 🌙 **Light & dark variants** — templates available in multiple color themes

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/damartripamungkas/awesome-login-web.git
   ```

2. **Open any template directly in your browser**
   ```bash
   cd awesome-login-web
   open login-food-bright.html
   ```

3. **Customize** the HTML/CSS to match your project's branding and backend.

No build step. No dependencies to install. Just open and go.

## 🤝 Contributing

Contributions are warmly welcome! If you've built a login page design you're proud of:

1. Fork this repository
2. Add your `.html` file to the root directory
3. Add a screenshot to `screenshots/`
4. Update the Live Demos table in this README
5. Open a Pull Request with a short description

Please make sure your template is responsive and uses clean, readable code.
